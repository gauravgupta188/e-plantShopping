e-plantShopping

# Project Overview

e-plantShopping is an e-commerce application designed for browsing and purchasing a wide variety of plants online. The application provides a simple and user-friendly shopping experience, allowing users to explore different plant categories, view detailed product information, add items to their cart, and manage their purchases efficiently.
